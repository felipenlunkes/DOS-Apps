[20/12/2015]: Fase de planejamento

- Início do planejamento de reestruturação de aplicativos.

[20/12/2015 - 20/01/2016]: Fase de desenvolvimento de código

- Lançamento da versão estável da nova biblioteca proprietária de UI.
- A nova API criada se mostrou estável.
- O uso da mesma nos aplicativos se mostrou viável.
- BINMOUSE portado para a nova API.
- PIN portado para a nova API.
- CPU portado para a nova API.

[13/02/2016]: Início das reestruturações efetivas

- BINMOUSE renomeado para MOUSE.
- Removido aplicativo PXDIAG (funções integradas ao PAINEL DE CONTROLE).
- PCINFO renomeado para PXDIAG.
- Removido aplicativo CPUID (funções integradas ao PAINEL DE CONTROLE).
- Removido aplicativo PROCVER (funções integradas ao PAINEL DE CONTROLE).
- Nome de fonte PXDESLIGAR.ASM alterado para DESLIGAR.ASM.
- Início da adequação do restante dos aplicativos para a nova API.
  SUB: VER, AJUDA, GUIA, CPU, DESLIGAR, INT90h, MEM, PXDIAG, REG, SOBRE, SOM, EDITOR
- Aplicativo APPX não sofrerá alterações significativas. Apenas para compatibilidade.
  SUB: linha de comando, sem interface; reconhecimento APPX.
- SOM portado para a nova API.
- SOM renomeado para AUDIO.
- DESLIGAR portado para a nova API.

[20/03/2016]: Progresso da reestruturação

- Bilioteca 'versao.s' adicionada ao SDK, permitindo ao programa exibir a versão do PX-DOS®.
- Removido aplicativo SOBRE (funções integradas ao PAINEL DE CONTROLE).
- Mudanças estruturais no aplicativo PAINEL DE CONTROLE.
  SUB: página de informações técnicas, exibição da real versão do sistema.

[13/06/2016]: Adequação a novas APIs e remoção de Apps desnecessários no momento
